#Introducing Falconism

- Ultra-minimalist blogging platform
- Tiny memory footprint
- Built-in SEO tools
- Embedded database
- Embedded webserver
- Packed in ONE tiny executable file

### Installation

You don't install it.
It's only ONE tiny executable file.

### On Linux :
```sh
$ ./falconism
```

### On Windows :
Just click falconism.exe 

OR,,

```sh
./falconism
```
